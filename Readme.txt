Blockchain-Based EHR
Using Python and Solidity
Access control based Electronic Health Record System

Requirements:
Python 2 or Python 3
Python Tkinter for web interface
Web3.py Pyhton Library
Metamask extension in Web Browser
Ganache

Steps:
Connect with local blockchain (Ganache) using web3 and metamask.
Deploy the Smart Contract and get the address to connect with Python and Ganache
Run the Python code, Register as Admin or Patient or Doctor.
Login as Admin and ADD records.
Login as Doctor and request for access by entering the Patient ID.
Login as Patient or Doctor and view records based on permission.
After any transaction is performed the block gets added into local blockchain.
